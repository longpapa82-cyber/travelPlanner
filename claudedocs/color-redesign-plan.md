# 여행 플래너 앱 색상 재설계 계획서

## 📌 현재 문제점
- **Primary Color**: Red (#FF6B6B) - 여행 앱에 부적합한 공격적인 색상
- 긴장감과 경고를 연상시키는 빨간색은 휴식과 즐거움을 추구하는 여행 경험과 불일치
- 브랜드 정체성이 여행의 편안함과 설렘을 전달하지 못함

## 🎯 설계 목표
1. **감성 정렬**: 여행의 자유로움, 모험, 휴식을 시각적으로 전달
2. **2025 트렌드 반영**: Earthy tones, Aqua & Sand, Mocha 무스 활용
3. **접근성 유지**: WCAG 2.1 AA 기준 충족
4. **다크모드 최적화**: 시간대별 자연스러운 전환

## ✅ 접근성 검증

모든 색상 조합은 다음 기준을 충족해야 합니다:

### 텍스트 대비율
- **일반 텍스트**: 최소 4.5:1
- **큰 텍스트 (18pt+)**: 최소 3:1
- **Primary 버튼**: #3B82F6 배경 + #FFFFFF 텍스트 = 4.8:1 ✅

### 색맹 고려
- Blue-Orange 조합은 색맹 사용자에게도 구분 가능
- Red-Green 조합 회피

## 🎯 기대 효과

1. **브랜드 정체성 강화**
   - 여행의 자유로움과 모험 전달

   - 눈의 피로 감소 (저채도 색상)

3. **2025 디자인 트렌드 선도**
   - Pantone, 업계 트렌드 반영
   - 경쟁 앱 대비 현대적 디자인

4. **전환율 향상 예상**
   - 적절한 CTA 색상 (Coral Pink)
   - 신뢰감 증대 (Ocean Blue)
