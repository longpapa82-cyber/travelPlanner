## 🚀 프로덕션 게시 절차

### Step 1: Google Cloud Console 접속

1. 다음 URL을 브라우저에서 열기:
   ```
   https://console.cloud.google.com/apis/credentials/consent?project=tripplanner-486511
   ```

2. Google 계정 로그인 (longpapa82@gmail.com 또는 프로젝트 소유자 계정)

3. 프로젝트 선택: **tripPlanner** 확인

---

### Step 2: 게시 상태 확인

현재 화면에서 다음 섹션을 찾기:
```
게시 상태
[테스트 중] 또는 [Testing]
```

**확인 항목**:
- "테스트 중" 배지 또는 "Testing" 상태
- 아래에 "프로덕션으로 게시" 또는 "PUBLISH APP" 버튼 존재

---

### Step 3: 프로덕션 게시 버튼 클릭

1. **"프로덕션으로 게시"** 또는 **"PUBLISH APP"** 버튼 클릭

2. 확인 대화상자가 나타나면 내용 읽기:
   ```
   앱을 프로덕션으로 게시하시겠습니까?

   프로덕션으로 게시하면 모든 Google 계정 사용자가
   귀하의 앱에 로그인할 수 있습니다.

   민감한 또는 제한된 범위를 요청하는 경우
   Google 검증이 필요할 수 있습니다.
   ```

3. **"게시"** 또는 **"PUBLISH"** 버튼 클릭하여 확인

---

### Step 4: 검증 요구사항 확인

#### TravelPlanner의 OAuth 범위
현재 앱이 요청하는 권한:
- ✅ `email` - 사용자 이메일 주소 (민감하지 않음)
- ✅ `profile` - 기본 프로필 정보 (민감하지 않음)

#### 예상 결과
- **즉시 승인**: 위 범위는 민감하지 않으므로 Google 검증 불필요
- **게시 상태**: "프로덕션" 또는 "In production"으로 즉시 변경
- **사용자 한도**: 제한 없음

#### 만약 검증 요구 시 (드물음)
```
검증 필요
귀하의 앱은 민감한 범위를 요청하므로
Google의 검증이 필요합니다.

예상 소요 기간: 1-6주
```

**조치**:
- 검증 프로세스 시작 (Google에서 이메일로 안내)
- 대안: 임시로 테스트 사용자 추가 (최대 100명)

## 🚨 트러블슈팅

### 문제 1: "앱이 확인되지 않음" 경고

**증상**:
```
이 앱은 확인되지 않았습니다
TravelPlanner에서 Google 계정에 액세스하려고 합니다

[돌아가기]  [고급] → [TravelPlanner(으)로 이동(안전하지 않음)]
```

**원인**:
- 브랜딩 정보 미완성 (앱 이름, 로고 등)
- 프로덕션 게시 직후 일시적 현상 (최대 24시간)

**해결**:
1. "고급" 클릭 → "TravelPlanner(으)로 이동" 클릭 (테스트용)
2. 브랜딩 정보 완성:
   - OAuth 동의 화면 → 브랜딩 탭
   - 앱 이름: MyTravel
   - 앱 로고: 업로드
   - 개인정보처리방침: https://mytravel-planner.com/privacy
   - 서비스 약관: https://mytravel-planner.com/terms

---

### 문제 2: "redirect_uri_mismatch" 에러

**증상**:
```
오류 400: redirect_uri_mismatch

리디렉션 URI가 OAuth 클라이언트 ID에 대해
승인된 리디렉션 URI와 일치하지 않습니다.
```

**원인**: Redirect URI 미등록

**해결**:
1. Google Cloud Console → API 및 서비스 → 사용자 인증 정보
2. OAuth 2.0 클라이언트: "TravelPlanner" 클릭
3. 승인된 리디렉션 URI에 추가:
   ```
   https://mytravel-planner.com/api/auth/google/callback
   http://localhost:3001/api/auth/google/callback
   ```
4. 저장 후 재시도

---

**증상**:

```
앱 검증 필요

예상 소요 기간: 1-6주
```

**장기 해결**:

- 검증 완료 후 자동으로 프로덕션 게시
