## 🎯 이번 업데이트 주요 내용

### 1. 사용자 동의 화면 추가 ✨
첫 로그인 시 **동의 화면**이 표시됩니다:
- ✅ **필수 동의** 2개: 서비스 이용약관, 개인정보 처리방침
- 📝 **선택 동의** 5개: 위치 정보, 푸시 알림, 사진 접근, 마케팅 수신 등

- 정책 업데이트 시 자동 재동의 요청

### 3. 사용자 경험 개선 🌟
- **전체 동의** 버튼으로 빠른 동의 가능
- 다크/라이트 모드 완벽 지원
- 한국어/영어 다국어 지원

## 🧪 테스트 시나리오

아래 7가지 시나리오를 순서대로 테스트해주세요. 각 시나리오마다 **결과를 체크**해주시면 감사하겠습니다.

### ✅ Scenario 1: 첫 로그인 (신규 사용자)
**목적**: 동의 화면이 정상적으로 표시되는지 확인

**단계**:
1. 앱 삭제 → 재설치 (또는 앱 데이터 삭제)
2. 앱 실행 → Google 또는 Kakao 로그인
3. **확인 사항**:
   - [ ] 로그인 후 **동의 화면**이 자동으로 표시되나요?
   - [ ] 7개 동의 항목이 모두 보이나요?
   - [ ] "서비스 이용약관", "개인정보 처리방침(필수)"에 **"필수"** 배지가 표시되나요?

**예상 화면**:
```
━━━━━━━━━━━━━━━━━━━━━
서비스 이용 동의
━━━━━━━━━━━━━━━━━━━━━
여행 플래너를 이용하기 위해
다음 약관에 동의가 필요합니다

[ ☑ ] 전체 동의

[ ☐ ] 서비스 이용약관 [필수]
[ ☐ ] 개인정보 처리방침(필수) [필수]
[ ☐ ] 개인정보 처리방침(선택)
[ ☐ ] 위치 정보 이용
[ ☐ ] 푸시 알림
[ ☐ ] 사진 접근
[ ☐ ] 마케팅 수신 동의

      [   확인   ]
━━━━━━━━━━━━━━━━━━━━━
```

---

### ✅ Scenario 2: 필수 동의 검증
**목적**: 필수 동의 없이 진행할 수 없는지 확인

**단계**:
1. 동의 화면에서 **선택 항목만** 체크 (필수 항목 체크 안 함)
2. **확인** 버튼 클릭
3. **확인 사항**:
   - [ ] "필수 약관에 동의해주세요" 에러 메시지가 표시되나요?
4. 필수 항목 2개 체크
5. **확인** 버튼 클릭
6. **확인 사항**:
   - [ ] "동의 정보가 업데이트되었습니다" 성공 메시지가 표시되나요?
   - [ ] 메인 화면(홈)으로 자동 이동하나요?

---

### ✅ Scenario 3: 전체 동의 기능
**목적**: "전체 동의" 버튼이 정상 작동하는지 확인

**단계**:
1. 동의 화면에서 **"전체 동의"** 버튼 클릭
2. **확인 사항**:
   - [ ] 7개 항목이 **모두 체크**되나요?
3. **"전체 동의"** 버튼 다시 클릭
4. **확인 사항**:
   - [ ] 7개 항목이 **모두 체크 해제**되나요?
5. "전체 동의" → "확인" 버튼 클릭
6. **확인 사항**:
   - [ ] 성공적으로 메인 화면으로 이동하나요?

---

### ✅ Scenario 4: 다크 모드 지원
**목적**: 다크/라이트 모드에서 UI가 정상 표시되는지 확인

**단계**:
1. 기기 설정 → 디스플레이 → **다크 모드 활성화**
2. 앱 재시작 → 동의 화면 진입
3. **확인 사항**:
   - [ ] 다크 모드에서 배경이 어두운 색상으로 표시되나요?
   - [ ] 텍스트가 밝은 색상으로 보이나요?
   - [ ] 가독성이 좋나요?
4. 기기 설정 → **라이트 모드로 전환**
5. 앱 재시작
6. **확인 사항**:
   - [ ] 라이트 모드에서 배경이 밝은 색상으로 표시되나요?
   - [ ] 텍스트가 어두운 색상으로 보이나요?

---

### ✅ Scenario 5: 다국어 지원
**목적**: 한국어/영어 번역이 정확한지 확인

**단계**:
1. 기기 설정 → 언어 → **한국어** 선택
2. 앱 재시작 → 동의 화면 확인
3. **확인 사항**:
   - [ ] "서비스 이용 동의", "전체 동의" 등 한국어로 표시되나요?
   - [ ] 번역이 자연스러운가요?
4. 기기 설정 → 언어 → **English** 선택
5. 앱 재시작 → 동의 화면 확인
6. **확인 사항**:
   - [ ] "Service Agreement", "Agree to All" 등 영어로 표시되나요?
   - [ ] 번역이 정확한가요?

---

### ✅ Scenario 6: 재로그인 (동의 완료 사용자)
**목적**: 이미 동의한 사용자는 동의 화면을 건너뛰는지 확인

**단계**:
1. Scenario 1-3 완료 후 **로그아웃**
2. 다시 **로그인**
3. **확인 사항**:
   - [ ] 동의 화면이 **표시되지 않고** 바로 메인 화면으로 이동하나요?

---

### ✅ Scenario 7: 네트워크 에러 처리
**목적**: 네트워크 에러 시 사용자 피드백이 제공되는지 확인

**단계**:
1. 기기를 **비행기 모드**로 설정
2. 동의 화면에서 "전체 동의" → "확인" 클릭
3. **확인 사항**:
   - [ ] "동의 정보 업데이트에 실패했습니다" 에러 메시지가 표시되나요?
4. 비행기 모드 **해제**
5. "확인" 버튼 다시 클릭
6. **확인 사항**:
   - [ ] 성공 메시지가 표시되고 메인 화면으로 이동하나요?
